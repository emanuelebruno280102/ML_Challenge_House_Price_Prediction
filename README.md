# ML_Challenge_Emanuele_Alvaro
Se trata del repositorio de nuestro trabajo sobre aprendizaje supervisado y no supervisado. Hemos descargado el kaggle "Housing Prices Competition Learn for Kaggle Learn Users" donde disponemos de 79 variables explicatorias que describen casi todos los aspectos de los hogares residenciales en Ames, Iowa.

El objetivo será probar la veracidad o la falsedad de si las negociaciones en el precio de una vivienda dependen de mucho más que del número de habitaciones. Para ello, seleccionamos varios modelos de Machine Learning y justificamos nuestra elección.
